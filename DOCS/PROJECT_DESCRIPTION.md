Automation, AI Agent workflows, and technical infrastructure for The Financial Club (TFC) — a curated, membership-based professional network for fintech leaders.

This project covers all n8n workflow design, Supabase database architecture, AI agent development, and system integration work for TFC's operations, including (but not limited to):

- Head of Sales Agent: Automated email outreach, follow-up sequences, CRM, and membership sales
- Event Operations: Campaign automation tied to TFC's event calendar (Connect Series, Retreats, Delegations, Cyber Connect, Fintech Fest)
- Membership Lifecycle: Onboarding, engagement tracking, upsell flows (Core → Plus → Premier)
- Platform Integration: Connecting the TFC member app (app.financialclub.com) with automation infrastructure
- Future Agents: Any new AI-driven automation roles for TFC operations

Client/Founder: Alex Pelin, Founder of The Financial Club
